Data Transfer Methods

Various data transfer methods in AppsFlyer and their characteristics

AppsFlyer Data Transfer Methods Overview

AppsFlyer provides marketing data in various ways. Each method is designed for different use cases and requirements.

Pull API

A method to request and retrieve data when needed

Push API

Receive data in real-time when events occur

Data Locker

An advanced solution for reliably transferring large volumes of data

Pull API

Everyday Analogy: Like calling to order food delivery. You request and receive data when you need it.

Pull API Characteristics

  • Request-Based: Call the API to retrieve data when needed.
  • Flexibility: Selectively retrieve only the data you need.
  • Batch Processing: Retrieve data for a specific time period at once.
  • Suitable Use Cases: Regular reports, dashboard updates, ad-hoc analysis

// Pull API Example Request

GET https://hq.appsflyer.com/export/app_id/installs_report/v5

?api_token=YOUR_API_TOKEN

&from=2025-01-01

&to=2025-01-31

&additional_fields=install_app_store,match_type,contributor1_match_type

For more details, refer to the Pull API Official Documentation.

Push API

Everyday Analogy: Like receiving notifications from your bank whenever deposits or withdrawals occur. You receive data in real-time as events happen.

Push API Characteristics

  • Real-time Transmission: Data is sent immediately when events occur.
  • Webhook-Based: Data is automatically sent to a specified endpoint.
  • Low Latency: Receive data almost in real-time.
  • Suitable Use Cases: Real-time dashboards, immediate marketing responses, fraud detection

// Event handling logic

if (eventData.event_name === 'install') {

// Handle install event

}

res.status(200).send('Event received');

});

For more details, refer to the Push API Official Documentation.

Data Locker

Everyday Analogy: Like a pipeline. A robust system that reliably and continuously supplies large volumes of data.

Data Locker Characteristics

  • Large Volume Data: Efficiently handles large datasets.
  • Cloud Integration: Directly integrates with cloud storage like AWS S3, Google Cloud Storage, and Azure Blob Storage.
  • Automatic Transfer: Automatically transfers data according to a set schedule.
  • Data Retention: Allows for long-term data retention and access.
  • Suitable Use Cases: Big data analysis, machine learning, advanced business intelligence

Data Locker is AppsFlyer's most powerful data transfer solution, making it the optimal choice for enterprise-grade data analysis.
