Data Locker Cohort

User behavior insights through advanced cohort analysis

What is Data Locker Cohort?

Data Locker Cohort is AppsFlyer's powerful cohort analysis tool that allows you to track and analyze user group behavior over time. This enables in-depth analysis of marketing campaign effectiveness, user retention, profitability, and more.

Differences from Traditional Cohort Analysis

Data Locker Cohort offers the following advantages compared to traditional dashboard or API-based cohort analysis:

Benefits of Data Locker Cohort

  • More Data: Access to more raw data than what's available in dashboards.
  • Flexible Analysis: Analyze data in any way you want using SQL queries.
  • Custom Metrics: Create customized metrics that fit your business needs.
  • Extended Time Analysis: Analyze data over longer periods.
  • Data Integration: Easily integrate with other data sources.
  • Automation: Automate analysis processes to generate regular reports.

Key Use Cases

LTV Analysis

Analyze customer lifetime value (LTV) by user acquisition channel to identify the most profitable marketing channels.

Retention Analysis

Compare retention rates across different user segments to identify churn causes and develop improvement strategies.

Campaign Performance Analysis

Measure the long-term performance of marketing campaigns to optimize ROI.

Use Case: LTV Analysis by Media Source

Here's an example of using Data Locker Cohort to analyze user LTV by media source.

Sample Data

Below is the sample data structure provided by Data Locker:

appsflyer_id install_date media_source campaign event_date event_name revenue
id123 2025-01-01 Facebook Spring_Sale 2025-01-05 purchase 15.99
id456 2025-01-02 Google Winter_Promo 2025-01-10 purchase 24.99
id789 2025-01-03 TikTok New_User 2025-01-15 purchase 9.99

SQL Query Example

-- 30-day LTV Analysis by Media Source

SELECT

media_source,

COUNT(DISTINCT appsflyer_id) AS users,

SUM(CASE WHEN DATEDIFF(event_date, install_date) <= 30 THEN revenue ELSE 0 END) AS revenue_30d,

SUM(CASE WHEN DATEDIFF(event_date, install_date) <= 30 THEN revenue ELSE 0 END) / COUNT(DISTINCT appsflyer_id) AS ltv_30d

FROM

app_events

WHERE

install_date BETWEEN '2025-01-01' AND '2025-01-31'

AND event_name = 'purchase'

GROUP BY

media_source

ORDER BY

ltv_30d DESC;

Analysis Results

The analysis results above show that Google ads have the highest 30-day LTV. Based on these insights, you can allocate your marketing budget more efficiently.

Getting Started

Follow these steps to get started with Data Locker Cohort:

  1. Activate Data Locker in your AppsFlyer account.
  2. Connect your cloud storage (AWS S3, Google Cloud Storage, etc.).
  3. Set up your data transfer schedule.
  4. Use SQL query tools (e.g., Amazon Athena, Google BigQuery) to analyze your data.
  5. Visualize your analysis results to derive insights.

For more details, refer to the Data Locker Cohort Official Documentation.
